# Quizzes Module

Manages quiz creation, distribution, and grading.

## Responsibilities
- Create quizzes
- Manage quiz questions
- Track student responses
- Calculate scores
- Generate quiz reports
- Link quizzes to subjects/resources

## Files Structure
- `create.js` - Quiz creation
- `questions.js` - Question management
- `responses.js` - Student response tracking
- `grading.js` - Automatic grading logic
- `reports.js` - Quiz analytics and reports
