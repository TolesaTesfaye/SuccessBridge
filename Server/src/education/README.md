# Education Module

Manages the hierarchical education structure.

## Hierarchy Levels
1. **Education Level** - High School, University
2. **Category** - Grade, Stream, Program
3. **Institution** - University, School
4. **Department** - Engineering, Science, etc.
5. **Subject** - Data Structures, Physics, etc.

## Responsibilities
- Create and manage universities
- Create and manage departments
- Create and manage subjects
- Create and manage grades/streams
- Maintain hierarchical relationships

## Files Structure
- `universities.js` - University management
- `departments.js` - Department management
- `subjects.js` - Subject management
- `grades.js` - Grade/Stream management
- `hierarchy.js` - Hierarchy validation and traversal
