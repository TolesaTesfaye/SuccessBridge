# Users Module

Manages different user types and their profiles.

## User Types
- **Student** - Access to resources, quizzes, recommendations
- **Admin** - Manage university/department resources
- **Super Admin** - Full platform control

## Responsibilities
- User profile management
- Role assignment
- User data retrieval
- User permissions

## Files Structure
- `students.js` - Student-specific operations
- `admins.js` - Admin-specific operations
- `superadmins.js` - Super admin operations
- `profile.js` - User profile management
